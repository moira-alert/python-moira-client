# 3.0.0

‼️ Remove underscore-prefixed fields that contained scheduling time settings:

- `_start_hour`
- `_start_minute`
- `_end_hour`
- `_end_minute`
- `_timezone_offset`
- `disabled_days`

Because they were duplicates of data from sched field.

Please use sched field for scheduling settings instead.
